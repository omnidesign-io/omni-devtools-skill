# Contributing

This repository is a home for evolving Omni Design's dev-only style exploration patterns as the workflow improves.

Prefer small, practical updates that make style testing easier. Keep `SKILL.md` focused on core patterns; document specific template details or script integrations in reference files.

Before releasing a change, validate that `SKILL.md` still has valid frontmatter and that the README accurately describes when the skill should be used.
